数据模型:Models/hello.model.php:

\RSystem\system::load_class('db' , '' , 0);//载入数据库操作类，(类名[，额外路径[，是否new]])
class hello extends db{
  public $PDO_OBJ = null; // PDO对象
  public function __construct() {
    $this->PDO_OBJ = \RSystem\system::load_pdo('', true); // (数据库名称_可以不填则为配置默认[,是否开启memcache])
  }

  public function search() {
    $ResArr = $this->PDO_OBJ->select_Tab('test')->select_Obj('*')->search_command(); // 查询test表内的所有*字段数据
    // var_dump($ResArr);
    return $ResArr;
  }
}

入门：上传文件 (以及关于URL以及Get参数)

视图Views/upload.html:

//在页面中访问  .../Index/index/testUploadView

form表单中的url说明:{{URL_HEAD}}/Home文件夹名/控制器名/方法名[&参数1[&参数2[&...]]]

此处class参数，指存储目录Storage下的指定文件夹，如果不指定的话则存储在Storage根目录下面,filename参数为文件保存的名称，如果不设置的话则为文件的默认名称

以上为关于Get参数的传递以及路径URL代码规范

<form action="{{URL_HEAD}}/Index/index/uploadFile" enctype="multipart/form-data" method="post" name="uploadfile"<
<input type="text" name="filename" /<<br<
<input type="text" name="class" /<<br<
<input type="file" name="textFile" /<<br<
<input type="submit" value="上传" /<</form<


控制器Controller/Index/index:

testUploadView是用于渲染上传文件html页面

uploadFile用于接收上传结果回调,$dataArr是回调结果返回的数据.

$dataArr['upload_error']元素是一个数组,存储报错文件和报错描述,如果没有的话则上传成功!(上传之前记得文件夹权限)

以上为关于Get参数的传递以及路径URL代码规范

// 上传文件html
public function testUploadView() {
  echo \RSystem\system::drawViews('upload');
}
// 上传文件结果回调
public function uploadFile($dataArr) {
  if (count($dataArr['upload_error']) <= 0) {
    echo "上传成功";
  } else {
    echo var_dump($dataArr['upload_error']);
  }
}
